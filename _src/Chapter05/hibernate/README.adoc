= Quarkus and Hibernate example

An example application featuring a One-To-Many relation between the table Customer and Orders

== How to run it

* Start the Database:

[source,shell]
----
$ ./db.sh
----

* Launch the application:

[source,shell]
----
$ mvn install quarkus:dev
----

* Deploy on Openshift:

[source,shell]
----
./deploy-openshift.sh
----
